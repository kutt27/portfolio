# Portfolio Website

This is a simple portfolio website with minimal design, showcasing various projects and skills.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-website
   ```

3. Open `src/index.html` in your web browser to view the portfolio.

## Features

- Responsive design
- Minimalist layout
- Sections for about, focuses, projects, and contact information
